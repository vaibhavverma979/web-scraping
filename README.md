# Web Scraper with AI-Powered Topic Extraction

A Python web scraping tool that uses BeautifulSoup and requests to extract text, images, and links from websites. Now includes AI-powered topic extraction using OpenAI or OpenRouter API.


**Quick Start (Web UI)**

pip install -r requirements.txt
python app.py
```

## Features

- **Text Extraction**: Search by keyword, CSS selector, or AI topic query
- **Image Extraction**: Search by keyword, list all images, or use CSS selector
- **Link Extraction**: Search by keyword or CSS selector
- **AI-Powered Topic Query**: Use natural language to find and extract specific sections
- **OpenRouter Support**: Use OpenRouter for access to multiple AI models (GPT-4, Claude, Gemini, etc.)
